الهدف

في صفحة عرض فاتورة المشتريات (Purchase Invoice View Page) عندي قسم كامل بعنوان "المورد" (كارت على اليمين/الشمال فيه الاسم/الهاتف/البريد/الرقم الضريبي…).

أنا عايز:

تشيل كارت/قسم المورد بالكامل من الـ UI (لا يظهر نهائيًا).

يبقى فقط اسم المورد موجود داخل كارت “التفاصيل” (معلومات الفاتورة/التفاصيل) كسطر واحد واضح.

بدون تغيير أي Backend / Queries / DB / RPCs — تغيير واجهة فقط.

يدعم RTL ويظهر بالعربي والإنجليزي بنفس نظام الترجمة الحالي.

نطاق التغيير (Files)

غالبًا الملف:

src/pages/purchasing/PurchaseInvoiceViewPage.tsx

(لو فيه Component منفصل لكارت المورد، عدّله/احذفه من الصفحة فقط بدون حذف الكومبوننت لو متستخدم في أماكن أخرى)

المطلوب بالضبط في الـ UI
A) إزالة قسم المورد

احذف/علّق Rendering الخاص بكارت المورد (Card/Section) بالكامل:

العنوان "المورد"

الحقول: الاسم، الهاتف، البريد، الرقم الضريبي، العنوان… إلخ

لا تترك مكان فاضي كبير أو layout مكسور.

B) إضافة اسم المورد داخل “التفاصيل”

داخل كارت "التفاصيل" (اللي فيه رقم الفاتورة، رقم فاتورة المورد، التاريخ، الحالة، الفرع…):

أضف سطر جديد:

Label عربي: "المورد:"

Label إنجليزي: "Supplier:"

القيمة: اسم المورد الحالي (مثلاً invoice.supplierName أو أي حقل متاح في الـ DTO).

لو الاسم غير موجود لأي سبب:

اعرض - (dash) بدل ما يوقع الصفحة.

C) تحسين الـ Layout بعد حذف الكارت

لو التصميم كان Grid بعمودين (Details + Supplier):

خليه عمود واحد بعرض كامل (Details تاخد Full Width).

لا تغيّر أي functionality تانية في الصفحة.

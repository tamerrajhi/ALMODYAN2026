DB-ONLY — Create Branch Accounts & Branch Sessions (NO CODE CHANGES)

Goal:
Add independent branch authentication model:
- branch_accounts: username/password for a branch (NOT linked to profiles/users)
- branch_sessions: active branch login sessions (cookie token hash lookup later)

Rules:
- DB changes only.
- Do NOT modify existing tables.
- Add constraints + indexes.
- Provide BEFORE/AFTER schema proof.
- Do NOT insert any default passwords.

Steps:

1) BEFORE:
SELECT to_regclass('public.branch_accounts') AS branch_accounts,
       to_regclass('public.branch_sessions') AS branch_sessions;

2) Migration (single transaction):
BEGIN;

CREATE TABLE IF NOT EXISTS branch_accounts (
  id uuid PRIMARY KEY DEFAULT gen_random_uuid(),
  branch_id uuid NOT NULL REFERENCES branches(id) ON DELETE CASCADE,
  username text NOT NULL,
  password_hash text NOT NULL,
  is_active boolean NOT NULL DEFAULT true,
  created_at timestamptz NOT NULL DEFAULT now(),
  updated_at timestamptz NOT NULL DEFAULT now(),
  UNIQUE (branch_id),
  UNIQUE (username)
);

CREATE INDEX IF NOT EXISTS branch_accounts_branch_id_idx ON branch_accounts(branch_id);

CREATE TABLE IF NOT EXISTS branch_sessions (
  id uuid PRIMARY KEY DEFAULT gen_random_uuid(),
  token_hash text NOT NULL UNIQUE,
  branch_id uuid NOT NULL REFERENCES branches(id) ON DELETE CASCADE,
  created_at timestamptz NOT NULL DEFAULT now(),
  expires_at timestamptz NOT NULL,
  revoked_at timestamptz,
  last_seen_at timestamptz
);

CREATE INDEX IF NOT EXISTS branch_sessions_branch_id_idx ON branch_sessions(branch_id);
CREATE INDEX IF NOT EXISTS branch_sessions_expires_at_idx ON branch_sessions(expires_at);

COMMIT;

3) AFTER schema proof:
SELECT to_regclass('public.branch_accounts') AS branch_accounts,
       to_regclass('public.branch_sessions') AS branch_sessions;

4) Columns proof:
SELECT column_name, data_type, is_nullable
FROM information_schema.columns
WHERE table_schema='public' AND table_name IN ('branch_accounts','branch_sessions')
ORDER BY table_name, ordinal_position;

STOP. Output the results.

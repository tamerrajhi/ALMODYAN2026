GOAL:
Gate /pos and /pos/return by POS operational session (__pos):
- If no POS session -> show Cashier select + PIN entry.
- If POS session exists -> render the POS page.
- Provide "Change Cashier" action that exits session and returns to gate.

API:
- GET  /api/pos/cashiers                -> list users for current branch
- POST /api/pos/cashier/enter           body {cashier_user_id, pin}
- POST /api/pos/cashier/exit
- GET  /api/pos/cashier/context         -> current cashier/session info (used for header)

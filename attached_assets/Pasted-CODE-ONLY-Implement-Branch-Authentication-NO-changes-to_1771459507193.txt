1) requireBranchSession(req,res,next)

   - if not found => 401 {error:"Branch session required"}

NEW ENDPOINTS:
A) POST /api/branch/login   (PUBLIC)
Body: { username, password }
Flow:
- find branch_accounts by username AND is_active=true
- bcrypt.compare(password, password_hash)
- on success:
  - create random token (32+ bytes) and store sha256(token) in branch_sessions.token_hash
  - expires_at = now() + interval '12 hours'
  - set cookie "__branch" = raw token (httpOnly, sameSite=lax, secure if production)
  - return {data:{branch_id}, error:null}
- on failure: 401 {error:"Invalid branch credentials"}

B) POST /api/branch/logout  (requireBranchSession)
- revoke current session: set revoked_at=now()
- clear cookie "__branch"
- return {data:{ok:true}, error:null}

C) GET /api/branch/context  (requireBranchSession)
- return {data:{branch_id:(req as any).branchId}, error:null}

ADD 3 ADMIN ENDPOINTS:

1) GET /api/admin/branch-accounts
- Returns list of branches with any linked branch_accounts info:
  { branch_id, branch_name, branch_code, has_account, username, is_active }
- username should be null if no account exists.
- Must NOT return password_hash.

2) POST /api/admin/branch-accounts/set-credentials
Body:
{ "branchId":"<uuid>", "username":"br1", "password":"Abcd1234", "isActive": true }
Validation:
- branchId required
- username required, trimmed, length 3..50
- password required, length >= 8 (enterprise minimum)
Behavior:
- bcrypt hash password
- UPSERT into branch_accounts:
  - branch_id UNIQUE
  - username UNIQUE
  - password_hash set
  - is_active = (isActive default true)
  - updated_at = now()
  - created_at = now() on insert

3) POST /api/admin/branch-accounts/revoke-sessions
Body:
{ "branchId":"<uuid>" }
Behavior:
- Set revoked_at = now() for all non-revoked sessions for that branch
- (Optional) also set expires_at = now() for immediate expiry
- Does NOT touch credentials

✅ PROMPT (Replit) — Add Excel Paste to Unique Items Grid + Reusable for Transfer
SCOPE (Strict)

UI فقط في هذه المرحلة: دعم Paste / Import من Clipboard إلى جدول الـ Grid.

لا تغيّر أي RPC أو DB في هذا البرومبت (لضمان صفر مخاطرة).

حفظ البيانات يستمر بنفس آلية الحفظ الحالية (RPC واحدة عند Save).

الهدف: المستخدم ينسخ من Excel ويلصق مباشرة في الجدول، يتكوّن صفوف تلقائيًا، ويظهر Validation قبل الحفظ.

المطلوب بالظبط

لما المستخدم يعمل Ctrl+V داخل أي خلية في الجدول:

لو الـ Clipboard يحتوي جدول (Tab-separated):

نفّذ parse إلى rows/cols.

لو لصق داخل صف موجود: حدّث حقول هذا الصف في الـ state.

لا تحفظ شيئًا تلقائيًا. فقط state.
